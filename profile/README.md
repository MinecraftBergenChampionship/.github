# Minecraft Bergen Championship (MBC)

We are a group of student developers that run a **private** Minecraft tournament.

**Disclaimer**

This tournament was initially inspired by, and serves as a partial recreation of, popular Minecraft tournaments such as [MCC](https://twitter.com/mcchampionship_), [BlockWars](https://twitter.com/BlockWarsEvent), and [Pandora's Box](https://twitter.com/PandorasBoxMC). We are not affiliated with these events, nor are we affiliated with Mojang or Minecraft, but we highly encourage anyone interested in Minecraft content to check out these events!
